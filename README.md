# Akıllı Otopark Asistanı

Bu proje, kullanıcıların bulunduğu konuma göre en yakın uygun otoparkları bulmasını amaçlayan basit bir web uygulamasıdır.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Özellikler
- Konum alma
- Yakındaki otoparkları listeleme
- Boş / dolu durumunu gösterme
- Durumu güncelleme
- Google Maps ile navigasyon başlatma

## Dosyalar
- `index.html` → Ana sayfa
- `style.css` → Tasarım dosyası
- `app.js` → Uygulama mantığı

## Çalıştırma
1. Dosyaları aynı klasöre koyun.
2. `index.html` dosyasını tarayıcıda açın.
3. “Konumumu Al” butonuna basın.
4. Otoparkları görüntüleyin.
